# Secure Login System

## Description
A secure login web application developed using Flask and SQLite.

## Features
- User Registration
- Secure Login with Hashed Passwords (bcrypt)
- SQL Injection Protection
- Session Management
- Logout Feature
- Optional Two-Factor Authentication (2FA)

## Technologies Used
- Python
- Flask
- SQLite
- Flask-Bcrypt

## Expected Outcome
A secure login system that protects user accounts using password hashing, input validation, session management, and secure authentication.
